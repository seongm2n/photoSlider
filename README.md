✍🏻프로젝트 개요 및 동기 

• 기억에 남는 사진을 모아놓은 웹페이지
• 클라우드처럼 실시간 저장할 수는 없지만 추억하고 싶은 사진 몇개를 슬라이드 기능을 통해 볼 수 있음
• 핸드폰 사진첩을 보다가 가끔 추억 꾸러미처럼 슬라이더로 보여주는 것이 보였음 
• 추억을 보여주는 기능을 웹페이지에도 적용할 수 있지 않을까 생각이 듦
